VanVan ブランドサイト
==================

ファイル構成:
------------
vanvan-site/
├── index.html          ← メインHTMLファイル
└── images/             ← 画像フォルダ
    ├── hero.jpg        ← ヒーローセクション背景
    ├── 01.jpg          ← ギャラリー画像1
    ├── 02.jpg          ← ギャラリー画像2
    ├── 03.jpg          ← ギャラリー画像3
    ├── 04.jpg          ← ギャラリー画像4
    ├── 05.jpg          ← ギャラリー画像5
    └── 06.jpg          ← ギャラリー画像6

使い方:
-------
1. index.htmlとimagesフォルダを同じ場所に配置
2. index.htmlをブラウザで開く

YouTube動画の設定:
-----------------
index.htmlの以下の部分を編集:
- VIDEO_ID_1 → 実際のYouTube動画ID
- VIDEO_ID_2 → 実際のYouTube動画ID

例: https://www.youtube.com/watch?v=dQw4w9WgXcQ
    → VIDEO_IDは「dQw4w9WgXcQ」
